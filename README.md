This was a project I completed for my Data Mining class. In this course, we used AWS for all of our work. The attached jupyter notebook was originally created and run in SageMaker in AWS.

The dataset I used for this project was the Fraudulent Transactions Prediction dataset from Kaggle. The dataset contains simulated (synthetic) financial transactions. There are 6,362,620 rows and 11 columns.

Kaggle Dataset: https://www.kaggle.com/datasets/vardhansiramdasu/fraudulent-transactions-prediction/data

In this project, I developed classification models to detect fraudulent transactions.
