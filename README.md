This was a project I completed for my Data Mining class. In this course, we used AWS. The attached jupyter notebook was originally created/ran in SageMaker in AWS.

The dataset I used for this project was the Fraudulent Transactions Prediction dataset from Kaggle. The dataset contains simulated (synthetic) financial transactions. The dataset has 6,362,620 rows and 11 columns.

Kaggle Dataset: https://www.kaggle.com/datasets/vardhansiramdasu/fraudulent-transactions-prediction/data
